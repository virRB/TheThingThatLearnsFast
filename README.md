# TheThingThatLearnsFast

## What is this?
* "TheThingThatLearnsFast" is a rewrite of the original neural network I made, but now optimised in C++

## How to use
* You input 4 pieces of data, each consisting of a, b, and an expected output.
* These can be logic gates or general patterns for the neural network to learn from
* It currently cannot learn XOR, as that is a limitation for one neuron neural networks like this
* After it has learnt, you can give it multiple inputs, where it will take those inputs and output a binary number based on the patterns it has learnt
